# Эпидемиология детской онкологии в России в 2022 году
## Синопсис
Данный проект посвящен комплексному анализу детской (0-14 лет) онкозаболеваемости в России за 2022 год. Основное внимание уделено выявлению региональных различий, возрастных и половых особенностей, а также определению наиболее часто встречающихся локализаций рака и аномалий (выбросов) в структуре заболеваемости. В ходе работы проведена очистка и агрегация данных, рассчитаны стандартизованные показатели на 100 тыс. детского населения, выполнена группировка регионов по уровню риска, выделены статистически значимые выбросы по возрастным группам. Для визуализации результатов построены плиточные карты и круговые диаграммы. Анализ основан на официальных данных Росстата и Российского Центра информационных технологий и эпидемиологических исследований в области онкологии (РЦИТЭО, МНИОИ им. П.А. Герцена).
## Актуальность
Злокачесвенные новообразования входят в число наиболее значимых заболеваний детского возраста, определяющих потребность в специализированной медицинской помощи. Изучение эпидемиологической ситуации необходимо для обоснованного планирования медицинской помощи и эффективного распределения ресурсов. Региональные различия в заболеваемости могут свидетельствовать о неравенстве в доступности диагностики и лечения. Анализ возрастно-половых паттернов позволяет выявить группы риска и определить приоритетные направления для оптимизации медицинской помощи детскому населению.
## Исследовательские вопросы

2) Каковы сходства и различия в заболеваемости между мальчиками и девочками?
3) Какие возрастные группы (0-4, 5-9, 10-14 лет) имеют наибольшую заболеваемость для разных локализаций заболеваний?
4) Какие регионы России имеют высокий, средний и низкий уровень детской онкологической заболеваемости?
5) Какие локализации являются выбросами (аномально высокими) в каждой возрастной группе?
6) Какова региональная дифференциация детской смертности от ЗНО в России в 2022 году?

### Процесс сбора и очистки данных:
1) Все исходные данные были выгружены из официального сборника РЦИТЭО (МНИОИ им. Пирогова), данные по численности населения из таблицы Росстата.
2) Для каждого набора данных создавался отдельный лист (листы _orig) в Google Sheets, впоследствии в копиях (листы с _edit) выполнялась очистка и расчет производных показателей.
3) Для каждого датасеты удалялись ненужные столбцы.
4) Унифицированы названия регионов, удалены суммарные значения для локализации заболеваний лимфатической и кроветворной тканей.
## Анализ
Анализ проводился в Google Sheets с последующей визуализацией в Datawrapper. Основные методы анализа:
1) **Стандартизованные показатели** на 100 тыс. детского населения для всех наборов данных (отдельно по полу, возрастным группам и суммарно). Это позволило корректно сравнивать абсолютные случаи.
2) **Выявление выбросов** по 95-му процентилю для возрастных групп (0-4, 5-9, 10-14). Локализации, превышающие порог, помечены как «Выброс». Это позволило выделить нозологии с аномально высокой заболеваемость в каждой возрастной группе.
3) **Группировка регионов по уровню заболеваемости/смертности** (низкий, средний, высокий) на основе 33-го и 67-го процентилей общего показателя заболеваемости/смертности. Результаты визуализированы на плиточной карте.
4) **Анализ структуры заболеваемости** через расчет долей (male_share%, female_share%). Построены диаграммы для мальчиков и девочек, показывающие распределение основных нозологий.
5) **Расчет rate_ratio_by_male** (соотношение заболеваемости мальчиков к девочкам) для выявления локализаций с половым дисбалансом.
6) **Топ-10 локализаций** по заболеваемости для мальчиков и девочек получены с помощью сводных таблиц по показателю на 100 тыс. детского населения по полу.
7) **Анитоп-10 регионов** заболеваемости/смертности сформированы через сводные таблицы по показателю на 100 тыс. детского населения в регионе.
8) **Локализации-выбросы по возрастным группам** по показателю на 100 тыс. детского населения каждой возрастной группы по 95-процентилю выделены через сводные таблицы.
